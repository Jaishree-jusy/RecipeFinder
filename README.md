# 🍽️ RecipeFinder

RecipeFinder is a full-stack MERN application that helps users discover delicious recipes using AI. Users can enter ingredients or recipe names, and the application generates recipe suggestions with easy-to-follow instructions.

---

## 🚀 Features

- 🔍 Search recipes using ingredients
- 🤖 AI-powered recipe generation using Groq API
- 📜 View recipe details and cooking instructions
- ⚡ Fast and responsive user interface
- 🌐 REST API integration
- 📱 Responsive design

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Jaishree-jusy/RecipeFinder.git
```

### 2. Navigate to the project

```bash
cd RecipeFinder
```

### 3. Install Backend Dependencies

```bash
cd backend
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

### 5. Create Environment Files

Create a `.env` file inside the `backend` folder and add:

```env
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run the Backend

```bash
cd backend
npm start
```

---

## ▶️ Run the Frontend

```bash
cd frontend
npm run dev
```
